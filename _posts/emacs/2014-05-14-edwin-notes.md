####C and M

- C(Control) ```ctrl```
- M(Meta) ```command```

####Screen navigation

- Move forward ```C-v```
- Move backward ```M-v```
- Clear screen ```C-l```

####Cursor navigation

- Previous line ```C-p```
- Next line ```C-n```
- Backward one character ```C-b```
- Forward one character ```C-f```
- Backward one word ```M-b```
- Forward one word ```M-f```

####Usually the Rule

- Meta commands are for English text
- Control commands are for basic textual units
